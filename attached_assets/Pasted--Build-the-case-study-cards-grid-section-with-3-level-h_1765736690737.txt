Typography Experimentation (Play Options)
Option 1: Font Weight Contrast

Label: 500 (medium)

Name: 700 (bold) OR 800 (extra bold)

Subtitle: 400 (normal)

Creates visual hierarchy through weight alone.

Option 2: Font Family Contrast (like hero)

Label: Same as body (sans-serif, system font).

Name: Different family (serif, or ultra-bold sans—e.g., 'Georgia' serif OR 'Inter' with letter-spacing tighter).

Subtitle: Same as body.

Example: If body is 'Segoe UI', name could be 'Georgia' or 'Playfair Display' (serif contrast).

Option 3: Size + Weight + Letter-Spacing

Label: 11px, 500, +0.05em spacing.

Name: 32px, 700, -0.02em spacing (tight, premium).

Subtitle: 15px, 400, 0em spacing (normal).

Pure hierarchy through size/weight/spacing.

Option 4: Mixed (Recommended)

Label: 11px, 500, color-muted, +0.05em spacing.

Name: 32px, 700 OR 800, color-dark, -0.02em spacing, italic accent OR different font-family (try serif name on sans body).

Subtitle: 15px, 400, color-secondary, normal spacing, possibly italic OR slightly lighter color.
